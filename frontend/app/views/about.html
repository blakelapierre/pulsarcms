About Pulsar getting better at helping people create

Pulsar is an open source project being distributed under the MIT license. It is a lot like other content management systems in that you can create an account on it and use it freely and you can host your own instance(s) to suit your needs to whatever scale is desired. Pulsar does scale quite happily from a hobby site stacked on one host to large, geographically diverse deployments. Pulsar is built atop the MEAN technology stack with a lot of additional libraries and frameworks helping it achieve trend-setting performance and service.

Pulsar began as a prototype that was quickly slapped together to prove some concepts I wanted to test. It is now becoming more formalized. A lot of code refactoring work has been done, and more will be happening soon. The front end is started to take final form for the default/reference client. The system supports robust user profile creation, basic content creation and a couple of fun bells and whistles thrown in. I am finishing Conversations now, and will be gettin back to/finishing Pulsar Studio after that.

As a project, Pulsar simply wouldn't be as far along as it is without the following open source projects available to support its needs so well. This isn't quite an audit, but here are some of the more prominent projects used by Pulsar to help its HTML5 presentation capabilities achieve best-of-breed results:

Software:
Node.js, MongoDB, memcached, GNU/Linux, Fedora and Ubuntu, GIMP, Audacity and the Three.js Editor (Beta).

Development Environment:
Fedora 20 hosting Sublime Text 2, Chrome and Firefox on the Cinnamon Desktop. I use Yeoman, Grunt, Bower and npm to scaffold in boilerplate code and manage the whole project.

I use git for revision control/configuration management with the git-cola GUI on Linux. The repository is hosted on github as an MIT-licensed open source project.

Karma, Jasmine and this whole great-big, gigantic list of tech that basically does not end - but all works. Somehow.

Linux Desktop Environments:
GNOME, KDE Plasma Workspaces and Cinnamon.

Web Services:
SoundCloud, YouTube, github.

Hosting:
PULSAR-ONE (my personal Pulsar instance) is hosted on a Dell Inspiron 530s, an old (discontinued) dual-core desktop PC with 2GB of RAM running Ubuntu on a rural DSL connection. The planned migration path is to Amazon Web Services and, after that, possibly its own data center some day. Hard to predict/don't care/it scales.

JavaScript Frameworks, SDKs and Tools:
AngularJS, Twitter Bootstrap, jQuery and jqlite, ExpressJS, Node.js API, MongooseJS, OpenSSL, socket.io, WebSockets, WebRTC, OpenGL ES 2.0 for the Web (WebGL), Three.js, HTML5 Audio (Web Audio, Media Stream Processing and Web Speech), CKEditor, ACE, Prism.js, Moment.js, and Shifty.
