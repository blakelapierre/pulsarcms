About This Site

I built this site from scratch using the MEAN stack (MongoDB, ExpressJS, AngularJS and Node.js). The front-end application is being served to the browser by Apache httpd on Ubuntu 12.04. The back-end API is powered by Node.js also running on Ubuntu 12.04.

This site is my personal blog, thought engine (because Twitter sucks) and technological playground. As I want to explore new technologies and learn them inside and out, I work on them here to build a new version of my personal site.
