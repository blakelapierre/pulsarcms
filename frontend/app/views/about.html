About Rob

I'm a freedom-loving individual living and working as a very frustrated web developer in Pittsburgh, PA. I've been a professional software developer since 1994, and I've been writing advanced, inventive software since I was a child. This has been a very long road for me.

I am a very inventive person. I have the ability to create extremely complex things quickly and from scratch. And, that's what I enjoy doing. From a prototype to bootstrapping a new development effort to answering a question in software for the very first time, it's simply what I enjoy doing. I genuinely enjoy making computers do everything.

Programming

In 4th grade, I began formally studying the BASIC programming language in my school's Gifted and Talented Education (GATE) program. By 7th grade, I had a video game entered into a state-wide competition. Between my junior and senior years of high school, I was selected as one of two students to attend Carnegie Mellon's Advanced Placement/Early Admission program where I studied the Pascal programming language in a University setting.

I've formally studied BASIC, Pascal, C, Assembly and COBOL (in that exact order). I've taught myself C++, Java, JavaScript, Lua, TCL, CSS and HTML. I have written full TCP/IP stacks that run in user space with their own ARP modules, etc., to simulate millions of "virtual" PCs using only 96 rack-mounted hosts. And, I've written systems that punch through consumer-grade NAT hardware to connect game players in peer-to-peer grids on PCs and video game consoles.

I used to run the OnDemand! BBS as part of the networks that predate the Internet and ran entirely on dial-up modems. I've built BBS "door" games in C on DOS. I've built systems that test very high capacity Cisco switches and routers. I've built the systems that power some of your favorite online games. And, I've built systems that simulate a full range of medical conditions experienced by a soldier at war in Iraq at Texas A&M University-Corpus Christi. I've written a smoothed particle hydrodynamics system to simulate actual human blood.

One of my side projects that's been on hold while focusing on Pulsar is GENERATOR, an actual software synthesizer built in JavaScript using the WebAudio API with an HTML5 single-page application front-end.

I wrote the system that served the web page you're reading right now. This isn't WordPress. This is Pulsar. It is my custom content management system. And, I built it because I'm tired of trying to extract the results I want from off-the-shelf CMS suites. I built Pulsar so it would just work, dammit.

Music

I've been creating music for as long as I've been disassembling my toys to make new ones and creating software. Much like software, the process of creating something amazing out of nothing is what music is all about for me. I love listening to other people's music as much as creating my own.

There, I learned all the ins & outs of working with professional audio equipment, and quickly became one of Pittsburgh's first MIDI recording engineers. To this day, I mostly work entirely in digital unless I'm not.
