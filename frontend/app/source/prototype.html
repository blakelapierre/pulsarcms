How the ?

Okay, this isn't all that complicated. The SoundCloud player that anyone can embed in any web page anywhere exposes a JavaScript API. PULSAR simply uses that API to retrieve information about tracks on SoundCloud inside an AngularJS controller. Then, it provides a whole new HTML5 experience by leveraging the information exposed by SoundCloud to web programmers.

AngularJS extends the HTML language with a whole new suite of elements, attributes and more that are intended to finally bring HTML up to par with modern web application design and development. By leveraging exactly these features along some clean best practices, I've been able to quickly and easily integrate all of the SoundCloud Widget API with AngularJS.

The end result: Your HTML can actually respond to a piece of media playing on the page using AngularJS directives such as ng-show, ng-click, ng-src and more. Have a read of the source code for this page. You will be quite shocked to see just how simple it really was to build this experience on Pulsar.

Please realize something: I am only getting started. This project is four weeks old, is being built on the MEAN stack and will soon be open sourced under the MIT license with a dowhatchalike policy. I've simply been very focused on achieving these goals with the software and genuinely just haven't taken the time to finish the documentation and get it online.

And, I would have more time to do that if I had some help. But, I'm doing this entirely on my own. That's not by choice. I've asked for help. I just haven't received any responses.
